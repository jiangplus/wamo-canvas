# Landing Page Assets

Place the following images in this `public/assets/` folder:

## Required Images:
- `matisse.png` - The dancers painting
- `cat_astronaut.png` - Cat astronaut illustration
- `flyer.png` - "Take what you need" flyer
- `right_hand.png` - Pointing hand illustration
- `left_hand.png` - Receiving hand illustration
- `phone.png` - Telephone illustration
- `popup_window.png` - "Spoiler Alert" popup window
- `key right arrow.png` - Right arrow key image
- `create key.png` - Create key image
- `together key.png` - Together key image
- `text on a string.svg` - Background vector decoration (SVG format)

## Notes:
- All images should be optimized for web (preferably under 500KB each)
- PNG format works best for illustrations with transparency
- The SVG file should be vector-based for crisp scaling
- Image sizes can vary but the component uses responsive classes to scale them appropriately
