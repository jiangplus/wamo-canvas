# Landing Page Fonts

Place the following font files in this directory:

## Required Fonts:
- `Arbotek-Light.otf` - Used for "the", "is just one big", "on an"
- `Handjet-Light.ttf` - Used for "universe"
- `Bygonest-Rustic-Regular.otf` - Used for "collage"
- `krifon-regular.otf` - Used for "infinite canvas"

## Font Usage:
- Arbotek-Light: Serif font for connecting text
- Handjet-Light: Sans-serif font for "universe"
- Bygonest Rustic Regular: Decorative font for "collage"
- Krifon Regular: Display font for "infinite canvas"
